## 🛡️ Declaração de Segurança de Dados (Data Safety Form)

Durante o questionário de privacidade obrigatório do Google Play Console, preencha os campos com base no seguinte mapeamento:

1. **O aplicativo coleta ou compartilha algum dado de usuário?**
   - *Sim. O aplicativo processa avaliações de terceiros e dados de perfil comercial para oferecer as análises de feedback.*
2. **Quais dados são compartilhados?**
   - *Finalidades Administrativas e de IA.* Dados de feedbacks e do comércio são transmitidos de forma segura para a API do Gemini, estritamente de maneira anônima e sob demanda, para elaborar sugestões de resumo de sentimento e posts sugeridos.
3. **Os dados coletados são criptografados em trânsito?**
   - *Sim. Todas as transferências de informações do LocalPulse utilizam conexões de canal HTTPS/SSL criptografadas de ponta a ponta.*
4. **O usuário pode solicitar a exclusão de seus dados?**
   - *Sim. O usuário pode desativar as contas ou realizar a limpeza imediata das preferências guardadas no banco de dados local limpando os dados gerenciados nas configurações padrão de armazenamento do sistema operacional Android.*

## 📸 Recomendações de Capturas de Tela (Screenshots Visual Assets)

Para a ficha de loja, o Google Play exige imagens nas seguintes dimensões padrão:

1. **Ícone de Alta Resolução:** PNG de 512 x 512, cantos normais de 32 bits, fundo transparente (limite de 1MB).
2. **Gráfico de Recursos (Feature Graphic):** Imagem promocional de 1024 x 500 JPG/PNG de 24 bits (composição visual do LocalPulse sobre fundo escuro).
3. **Capturas de Tela do Dispositivo (Móvel):**
   - Envie pelo menos 4 capturas com proporção 16:9 vertical de telas do celular (Onboarding, Painel da Home com indicador Offline, Avaliações Filtradas por IA, e Detalhe de Resposta Inteligente).
